# Myfistprogect
Action ou Vérité en ligne , crée un compte, choisis un adversaire et amuse-toi avec tes amis ! 
Action ou Vérité — Version Simple

Description

Ce projet est un jeu d’Action ou Vérité simple conçu pour iPad ou tout navigateur moderne. Les utilisateurs peuvent créer un compte (pseudo), se connecter et jouer avec d’autres joueurs en mode “hot-seat” (passer l’appareil au joueur suivant).

Le jeu génère aléatoirement des actions ou des vérités à relever et permet de copier les défis pour les partager.

Fonctionnalités
	•	Création de compte simple (pseudo uniquement).
	•	Connexion / Déconnexion.
	•	Liste des utilisateurs enregistrés (stockés localement dans le navigateur).
	•	Choix d’un adversaire.
	•	Génération aléatoire de vérités et d’actions.
	•	Bouton pour passer au prochain défi.
	•	Copier le défi dans le presse-papiers.
	•	Stockage local via localStorage (aucun serveur nécessaire).

Installation / Utilisation
	1.	Ouvrir Koder ou tout éditeur de texte sur iPad.
	2.	Créer un nouveau fichier et copier le contenu de action_ou_verite.html.
	3.	Sauvegarder le fichier.
	4.	Ouvrir le fichier dans Safari (recommandé) pour profiter pleinement de toutes les fonctionnalités.

Notes
	•	Le jeu est local uniquement : pas de connexion internet nécessaire.
	•	Pour une version multi-joueurs en ligne, un serveur et une base de données seraient nécessaires.
	•	Le code est entièrement écrit en HTML, CSS et JavaScript.
